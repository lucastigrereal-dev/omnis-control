## 2. TIPOGRAFIA SUGERIDA (GOOGLE FONTS)

### Familia Principal — Headlines e Destaques

**Playfair Display** (Google Fonts — gratuita)
- **Peso Bold (700):** Títulos principais, números de impacto (slide 3 "R$ 18.750")
- **Peso Regular (400):** Subtítulos elegantes
- **Carater:** Serifada classica com toque contemporaneo. Transmite elegância, tradicao, hotelaria premium.
- **Link:** `https://fonts.google.com/specimen/Playfair+Display`

### Familia Secundaria — Corpo de Texto e Bullets

**Inter** (Google Fonts — gratuita)
- **Peso SemiBold (600):** Subtítulos, chamadas de secao
- **Peso Regular (400):** Corpo de texto, bullets, notas
- **Peso Light (300):** Notas de rodape, fontes de dados
- **Carater:** Sans-serif geometrica, maxima legibilidade em telas pequenas. Neutra o suficiente para nao competir com a Playfair Display.
- **Link:** `https://fonts.google.com/specimen/Inter`

## 3. ESTILO DE FOTOGRAFIA

### Diretrizes Gerais
- **Estilo:** Documental sofisticado — parece foto real de hospede, nao foto de catalogo de hotel.
- **Iluminacao:** Luz natural sempre. Golden hour (amanhecer/entardecer) para exteriores. Luz difusa de janela para interiores.
- **Paleta cromatica das fotos:** Tons terrosos, verdes naturais, azuis suaves. Evitar saturacao artificial.
- **Tratamento:** Contraste medio-baixo, ligeiro warm shift (+5% temperatura), sem filtros Instagram obvios.
- **Proporcao:** 1:1 (quadrado) — corte pensado para feed, nao para galeria.

## 4. COMPOSICAO DOS SLIDES

### Grid Base — 1080x1080px

- **Margem de seguranca interna:** 80px em todos os lados (area util: 920x920px)
- **Margem de seguranca externa:** 20px (elementos criticos nao devem encostar na borda — corte do Instagram)
- **Grid invisivel:** 6 colunas x 6 linhas (cada celula ~153px) para alinhamento consistente

### Estrutura de Cada Slide

```
[1080px largura]
┌─────────────────────────────┐
│ ←Margem superior 80px       │
│                              │
│    TÍTULO (2 linhas max)     │  ← Altura: ~25% do slide
│                              │
│    ┌──────────────────┐      │
│    │  CORPO / DADOS   │      │  ← Altura: ~55% do slide
│    │  (bullets, icons, │      │
│    │   gráficos, mock) │      │
│    └──────────────────┘      │
│                              │
│     nota de rodapé           │  ← Altura: ~10% do slide
│ ←Margem inferior 80px       │
└─────────────────────────────┘
          ↑ Margem esquerda 80px
                         Margem direita 80px ↑
```

### Regras de Composicao

1. **Espaço negativo abundante.** Nunca preencher mais de 65% da area do slide com elementos visuais + texto.
2. **Ponto focal unico por slide.** O olho deve saber exatamente onde pousar primeiro.
3. **Progressao em Z.** Elemento principal no topo → fluir diagonalmente → fechar no canto inferior oposto.
4. **Consistencia de posicionamento.** Se o titulo esta no topo no slide 2, deve estar no topo no slide 7. Nao reinventar layout a cada slide.
5. **Contraste de fundo.** Alternar slides escuros (2, 3, 6, 8, 10) e claros (4, 5, 7, 9) para manter ritmo visual. Capa (1) é intermediario.

## 5. ICONES E ILUSTRACOES

### Estilo de Icones

- **Estilo:** Line icons (traco) com espessura de 2px. Preenchimento apenas em elementos de destaque.
- **Biblioteca sugerida:** Phosphor Icons (`phosphoricons.com`) — gratuitos, consistentes, mais de 1.200 opcoes.
- **Tamanho padrao:** 48x48px para icones em destaque, 32x32px para icones em bullets.
- **Cor padrao:** Herda a cor do texto do slide. Icones de destaque usam cor de acento (amarelo, verde, laranja).

- **Nao usar ilustracoes genericas estilo "corporativo flat".** Prefira fotografia real ou graficos de dados.
- Se necessario usar abstracao visual: formas geometricas simples (circulos, linhas, arcos) na paleta principal.

## 7. TOM VISUAL GERAL

### Tres Palavras que Definem o Tom Visual

1. **Clean** — Layout arejado, sem excessos, tipografia respira. Linhas limpas, grids invisiveis.
2. **Profissional** — Dados com fonte, hierarquia clara, consistencia cromatica entre slides.
3. **Aconchegante** — Paleta com tons quentes (bege, amarelo ouro, laranja suave) equilibrando a frieza do azul marinho. A sensacao e de "consultor que entende hotelaria", nao de "agencia de marketing generica".

### O Que EVITAR Visualmente

- ❌ Gradientes neon ou muito saturados
- ❌ Fontes script/cursivas (ilegiveis no mobile)
- ❌ Mais de 2 fontes diferentes no carrossel
- ❌ Emojis dentro dos slides (quebra a seriedade profissional)
- ❌ Fotos de banco de imagem genericas (maos apertando, pessoas em reuniao fake)
- ❌ Fundo branco puro em slides consecutivos (cansa a vista no mobile)
- ❌ Texto centralizado em mais de 3 linhas (dificulta leitura no mobile)
- ❌ Logos de OTAs (Booking, Expedia) visiveis — evitar questoes legais e de marca

### Mockup Mental — Como o Carrossel Deve "Respirar"

```
SLIDE CLARO (areia/branco) — pausa, respiro
SLIDE ESCURO (azul/preto)  — impacto, dado
SLIDE CLARO (areia/branco) — leveza, storytelling
SLIDE ESCURO (azul)        — autoridade, numero
SLIDE CLARO (branco)       — solucao, esperanca
SLIDE ESCURO (verde)       — case, confianca
SLIDE CLARO (branco)       — framework, clareza
SLIDE ESCURO (vermelho)    — alerta, urgencia
```
