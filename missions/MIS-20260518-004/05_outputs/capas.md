# Capas para Reels -- Feed Thumbnails

## Regra Geral das Capas
- 1 a 3 palavras MAXIMO no texto
- Rosto com emocao (surpresa / serio / sorriso) OU imagem do hotel parceiro
- Cores: paleta Brasil -- azul marinho, amarelo areia, verde natureza, branco
- A capa vende o CLIQUE. Nao explica o conteudo -- gera curiosidade.

## Checklist Rapido Antes de Postar Cada Capa

- [ ] Texto visivel no thumbnail pequeno (feed grid 3 colunas)
- [ ] Rosto do Lucas (reconhecimento de criador)
- [ ] Sem texto na borda inferior direita (icone de Reel do Instagram cobre)
- [ ] Contraste suficiente (testar: desfocar a imagem -- ainda da pra ler?)
- [ ] Consistencia visual (paleta das 10 capas forma um conjunto harmonico no feed)
- [ ] Nao repete layout -- cada capa tem identidade propria
