# Textos de Tela -- Overlay Scripts para Cada Reel

## Regras Gerais
- Maximo 4 palavras por tela
- Ritmo rapido (troca a cada 1-2 segundos)
- Fonte: sans-serif bold (Montserrat Bold ou Inter Extra Bold)
- Cor: branco com sombra preta (drop shadow) ou amarelo #F5A623 sobre fundo escuro
- Animacao: pop-in (escala 0% -> 110% -> 100% em 0.3s)
- Estilo: Mr. Beast / TikTok viral adaptado para turismo

## Padrao Visual dos Textos

| Elemento | Especificacao |
|---|---|
| **Fonte primaria** | Montserrat Extra Bold (titulos), Montserrat Semi Bold (apoio) |
| **Cor primaria** | Branco #FFFFFF |
| **Cor de destaque** | Amarelo #F5A623 |
| **Cor de alerta** | Vermelho #E63946 |
| **Cor de resultado** | Verde #2A9D8F |
| **Drop shadow** | 0px 4px 12px rgba(0,0,0,0.6) |
| **Stroke (borda)** | 3px preto quando texto esta sobre imagem clara |
| **Animacao padrao** | Pop-in (0.3s, cubic-bezier(0.68, -0.55, 0.27, 1.55)) |
| **Alinhamento** | Centro ou centro-inferior (nunca topo sem motivo) |

## Regras de Ouro -- Texto na Tela

1. **NUNCA repita o que o Lucas ja esta falando** -- o texto reforca, nao duplica
2. **1 ideia por tela** -- se precisa de 2 linhas, quebre em 2 telas
3. **Fonte grande** -- se nao da pra ler a 50cm do celular, ta pequeno
4. **Aparece e some** -- texto nunca fica estatico mais de 2 segundos
5. **CTA sempre com seta ou emoji** -- 👇 apontando pra area de comentario
